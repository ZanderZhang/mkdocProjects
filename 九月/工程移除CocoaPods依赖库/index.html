  • 工程移除CocoaPods依赖库

点这里--->CocoaPods安装和使用教程


当我们工程安装很多第三方开源类库,有时候不需要时,我们可以将其移除.
在stackoverflow发现How to remove CocoaPods from a project?


第二个answer比较简单

需要用到两个CocoaPods plugin

  • cocoapods-deintegrate
    • 安装gem install cocoapods-deintegrate
    • 运行pod deintegrate

这一步之后就剩下

Podfile
Podfile.lock
Workspace

  • 可以手动移除剩下的这三个,也可以用到cocoapods-clean
    • 安装gem install cocoapods-clean
    • 运行pod clean
    • 最后运行rm Podfile移除Podfile

如果还存在问题,可以把之前用到的库的路径移除,

项目目录下,在你的终端就是这样的顺序: 

gem install cocoapods-deintegrate
pod deintegrate
gem install cocoapods-clean
pod clean
rm Podfile

最后你可以在Podfile添加新的库使用,或不使用CocoaPods,当然还有Carthage.

记得备份你的项目!
